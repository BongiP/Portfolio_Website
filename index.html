Portfolio Projects

Portfolio

Location Finder App

A location finder app made with React that uses an API from Google Maps. Used to match the distance between two locations and how long it will take from to walk or to transit from one location to the other. It can also provide users with more contextual information than a street address or set of coordinates. I used the useState hook to store the current location of the user and the destination location. I also used the useEffect hook to fetch data from the Google Maps API when the component mounts.

Budget App

A budget application made with React. Uses React Router for routing and React Context API and Hooks for state management. The application has a log in and signup page that uses firebase authentication and routing to open the budget page upon successful login. It is a company budget allocation app, the budget can be adjusted according to the administrator's wishes.

Shopping App

An e-commerce app built with React that allows users to search for products, add items to their cart and checkout. The app uses states, hooks and simple react page to provide a seamless user experience. The app is designed to be fast, responsive and easy to use

Card Game

The game made with vanilla Javascript is a memory game where the player has to predict where a card is located. There are four cards, they shuffle and then the player has to peak. There are four games to play for each round and it records a score. Another feature of the game is that it can start where you left it when you come back again to play.
